# Bond.az - Elite SEO & Semantic Structure Strategy

Oxu.az-ın uğurlu SEO strukturu analiz edilərək, **Bond.az** üçün daha təkmil, müasir və aqressiv (max keyword growth) bir modelə çevrilmişdir. Bu strategiya Google-un 2026-cı il üçün EEAT (Expertise, Experience, Authoritativeness, Trustworthiness) meyarlarına tam uyğundur.

## 1. Texniki SEO Metadataları (Ana Səhifə)

| Element | Bond.az Standartı | Məqsəd |
| :--- | :--- | :--- |
| **Title Tag** | Bond.az - Son Xəbərlər, 24/7 Azərbaycan və Dünya Xəbərləri | Keyword Density |
| **Description** | Bond.az xəbər portalı: Ən son Azərbaycan və dünya xəbərləri, siyasət, iqtisadiyyat, texnologiya və idman hadisələri. Operativ və tərəfsiz xəbər mənbəyiniz. | CTR Optimizasiyası |
| **Hreflang** | `az`, `en`, `ru` (Hər dildə çarpaz linklər) | Qlobal İndeksləmə |
| **Canonical** | Dinamik HTTPS URL (Trailing slash-siz) | Duplicate Content-in qarşısını almaq |

## 2. Semantic Schema.org (JSON-LD) - Mükəmməl Struktur

Axtarış motorlarının (Google, Bing) saytı "anlaması" üçün aşağıdakı sxemlər hər dildə tətbiq olunur:

### A. NewsMediaOrganization Schema
Saytın real və etibarlı xəbər təşkilatı olduğunu təsdiqləyir.
- **Parametrlər:** Logo, Sosial Media linkləri, Əlaqə nömrələri, Redaksiya ünvanı.

### B. FAQPage Schema (Rich Results)
Google axtarış nəticələrində saytın altına sual-cavab bölməsi əlavə edir (Sizə gələn trafikin 30% artmasına səbəb olur).
- **Suallar:** Bond.az-da hansı xəbərlər var?, Xəbərlər nə vaxt yenilənir?, Mobil tətbiqi varmı? və s.

### C. ItemList Schema (Latest News Carousel)
Ana səhifədəki ən son 12 xəbəri Google-a "Carousel" şəklində təqdim edir.

### D. BreadcrumbList Schema
Sayt daxili iyerarxiyanı (Ana səhifə > Kateqoriya > Xəbər) Google-a izah edir.

## 3. Sosial Media Optimizasiyası (OpenGraph & Twitter Cards)

Hər xəbər paylaşıldıqda şəkillərin 1200x630 ölçüdə mükəmməl görünməsi üçün:
- `og:image:width: 1200`, `og:image:height: 630`
- `twitter:card: summary_large_image`

## 4. İnteqrasiya Planı (Timeline)

1. **Phase 1 (Həftə 1):** Ana səhifə metadata və Sxemaların (Organization, FAQ) tətbiqi.
2. **Phase 2 (Həftə 2):** Xəbər detallarında `Article` və `Speakable` (Səsli xəbər üçün) sxemalarının qurulması.
3. **Phase 3 (Həftə 3):** Kateqoriya səhifələrinin `CollectionPage` olaraq optimallaşdırılması.
4. **Phase 4 (Həftə 4):** XML Sitemap-lərin hər dildə avtomatik yaradılması və Google Search Console-a təqdim edilməsi.

---
**Nəticə:** Bu struktur tətbiq olunduqdan sonra, 1 ay daxilində xəbərləriniz Google-un "Top Stories" (Əsas xəbərlər) blokunda görünməyə başlayacaq.
